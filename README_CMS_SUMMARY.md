## 🎯 How to Use Your CMS

### **Access It:**
1. Go to `http://localhost:5173/login` (or your domain/login)
2. Email: `admin@guidingstars.com`
3. Password: `password123`
4. Click "Content" in the sidebar

### **Manage Content:**
1. Select a section tab (Hero, About, Testimonials, Footer, etc.)
2. Click "Add Content" or "Edit" on existing item
3. Fill in:
   - **Key** (unique ID like `hero_title`)
   - **Title** (display name like "Hero Title")
   - **Value** (actual content)
   - **Page** (which page it appears on)
4. Click "Save"

### **Upload Images:**
1. Click "Upload Image" button
2. Select file (5MB max, JPEG/PNG/WebP/GIF)
3. Copy filename to content field
4. Save

### **See Changes:**
- Changes appear immediately on public pages
- No restart needed
- No code deployment needed
- Real-time updates! 🚀
