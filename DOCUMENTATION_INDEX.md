## ❓ FAQ - Quick Answers

**Q: Can clients break the website with CMS updates?**  
A: No. CMS only updates content, cannot modify code or structure.

**Q: Do photos need to be exact dimensions?**  
A: No. CSS handles responsive sizing. Phase 2 will auto-resize.

**Q: How many text fields can I add?**  
A: Unlimited. Currently 65+, can add more anytime.

**Q: Can I add new pages with CMS?**  
A: No. New pages require code. CMS updates content only.

**Q: What if client accidentally deletes text?**  
A: Database backups daily. Can restore from backup.

**Q: Can multiple people edit at once?**  
A: Yes, but changes may conflict. Best practice: one at a time.

**Q: How much training does client need?**  
A: Very little. 30-minute walkthrough usually sufficient.

**Q: What if client uploads wrong format?**  
A: System validates. Only JPG, PNG, GIF, WebP allowed.
