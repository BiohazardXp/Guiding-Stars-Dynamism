## How to Upload & Replace Photos

### Step 1: Access the Admin Panel
- Go to `http://guiding-stars.com/content`
- Log in with your admin credentials

### Step 2: Find the Photo Field
Look for fields labeled:
- "Hero Banner Photo"
- "Team Member Photo"
- "Testimonial Photo"
- "Service Icon"
- "Event Photo"
- etc.

### Step 3: Upload New Photo
- Click "Upload Photo" button
- Select image from your computer
- Photo must be:
  - **Format:** JPG, PNG, GIF, or WebP
  - **Size:** Less than 5MB
  - **Dimensions:** See recommended sizes below

### Step 4: Save & Publish
- Click "Save"
- Changes appear on website immediately
- No waiting, no code deployment needed

## Photo Tips for Best Results

### ✅ Do's
- ✅ Use high-quality images
- ✅ Resize to recommended dimensions before uploading
- ✅ Use professional photos
- ✅ Keep file size under 5MB
- ✅ Use JPG or PNG format for best compatibility
- ✅ Add descriptive filenames (e.g., `john-doe-mentor.jpg`)

### ❌ Don'ts
- ❌ Don't upload very large files (>5MB)
- ❌ Don't use low-resolution images
- ❌ Don't use uncommon formats
- ❌ Don't upload blurry or pixelated photos
- ❌ Don't use photos with wrong dimensions (will look stretched)

## Troubleshooting

### Problem: Photo won't upload
**Solution:**
- Check file size (must be < 5MB)
- Try JPG or PNG format
- Clear browser cache and try again
- Contact support if issue persists

### Problem: Photo looks stretched or distorted
**Solution:**
- Resize photo to recommended dimensions
- Use photos with correct aspect ratio
- Example: For 300×300px, use a square photo

### Problem: Photo took too long to upload
**Solution:**
- Check internet connection
- Try a smaller file size
- Try a different format (JPG instead of PNG)
- Contact support if problem persists

### Problem: Changes not appearing on website
**Solution:**
- Wait 30 seconds for cache to clear
- Refresh browser (Ctrl+R or Cmd+R)
- Clear browser cache and refresh
- Contact support if problem persists
