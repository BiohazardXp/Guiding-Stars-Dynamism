# Quick Answer: Why Can't I Update Some Parts?

## Short Answer (30 Seconds)

Your website has **2 types of content:**

1. **Text & Photos** (✅ You can change via CMS)
2. **Structure, Layout, Colors** (❌ Locked in code, needs developer)

---

## Medium Answer (5 Minutes)

### What's in the CMS (✅ You Can Change)
- All text on the website (65+ fields)
- All photos and images
- Team member information
- Testimonials
- Contact information
- Footer content

**Access:** http://yourdomain.com/content  
**Time:** 2-5 minutes per update  
**Cost:** $0  
**Risk:** None (can't break anything)

### What's in the Code (❌ Needs Developer)
- Page layout & sections
- Navigation menu
- Colors (#FF9148 orange, etc.)
- Fonts & typography
- Form fields
- Features & functionality
- URLs & routing

**Time:** 1-14 days  
**Cost:** $200-5000+  
**Risk:** Medium (needs testing)

---

## Why Is It This Way?

**Safety:** Structure is locked in code to prevent breaking the website.  
**Performance:** Code runs faster than retrieving everything from database.  
**Security:** Sensitive logic must be protected in code, not exposed in CMS.  
**Stability:** Layout changes need careful testing before going live.

## Examples

### ✅ CAN Change (CMS)
- Update hero title: "Bridging Academia" → "Empowering Leaders"
- Replace team photo: old.jpg → new.jpg
- Update service description: "We offer..." → "We provide..."
- Add testimonial: "Jane Doe - Amazing program!"
- Change contact email: old@example.com → new@example.com

**Time:** 1-5 minutes each  
**Cost:** $0  
**Process:** Login to CMS → Edit → Save → Live instantly

### ❌ CANNOT Change (Code)
- Change orange color to purple
- Add "Services" to main menu
- Change "About" URL to "Our Story"
- Add search functionality
- Change page layout (2 columns → 3 columns)
- Add new form field to contact form
- Make photos auto-resize

**Time:** 1-14 days  
**Cost:** $200-5000+  
**Process:** Developer codes → Rebuilds → Tests → Deploys

## Key Insight: The 80/20 Rule

**80% of your updates:**
- Text changes ✅ (CMS, instant, $0)
- Photo updates ✅ (CMS, instant, $0)

**20% of your updates:**
- Design/layout ❌ (Code, days, $$$)
- New features ❌ (Code, weeks, $$$)
- Navigation changes ❌ (Code, hours, $$$)

**The CMS handles the 80% you need regularly.**
